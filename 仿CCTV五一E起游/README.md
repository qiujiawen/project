# 仿CCTV五一E起游

这是一个 以 CCTV五一E起游 为例的 react 开发项目。

## 技术栈

1、利用create-react-app搭建官方脚手架。

2、利用react构建用户界面利用react构建用户界面 。

3、利用rem对各种手机型号适配。

4、使用css3设置每个页面进场动画和出场动画、模糊设置、波纹动画制作。

5、使用DOM操作，对上传数据类型验证，判断上传的是图片还是视频。

## 项目脚本命令

在项目目录中，您可以运行：

### `npm start`

在开发模式下运行应用程序。

打开http：// localhost：3000地址并在浏览器中查看。
